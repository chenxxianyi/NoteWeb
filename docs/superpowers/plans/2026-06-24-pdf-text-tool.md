# PDF 文本批注工具实施计划

## 目标

在 PDF 阅读页工具栏新增“文本”工具，支持在页面上添加可编辑文本批注，并让文本批注可重新选中、移动、调整宽度、重新编辑和删除。

## 实施步骤

1. 扩展批注类型
   - 在 `pdfDrawingTypes.ts` 增加 `TextDrawing`，扩展 `Drawing` 和 `PDFActiveTool`。
   - 增加文本对象比较/命中所需的基础类型支持。
   - 验证：新增或扩展单元测试覆盖 text drawing equality。

2. 增加文本几何与渲染支持
   - 增加文本换行、边界计算、命中测试、移动和宽度调整辅助函数。
   - `PDFPageWrapper.vue` 的 canvas 渲染支持绘制多行文本。
   - 验证：单元测试覆盖文本换行、边界、命中、移动和 resize。

3. 增加文本编辑交互
   - 文本工具点击页面时创建 DOM `textarea` 编辑器，保存后写入批注。
   - 选择工具支持点击文本、拖动移动、拖右下角调整宽度、双击编辑、Delete 删除。
   - 橡皮擦忽略文本，文本删除走选择工具删除。
   - 验证：类型检查和构建。

4. 持久化和撤销
   - `PDFViewer.vue` 保存/加载 `tool: "text"` 的 `position_data`。
   - 文本编辑复用批量替换接口进入 edit history；删除走 erase history 以便撤销恢复。
   - 验证：单元测试、构建。

5. 工具栏接入
   - `ReaderView.vue` 添加文本按钮和字号滑杆。
   - 文本工具复用颜色选择，字号独立于画笔粗细。
   - 验证：构建通过；不运行 Playwright/E2E，除非用户另行要求。
