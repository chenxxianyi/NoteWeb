笔记编辑器工具栏增强方向

参考 Typora，但控制复杂度：增强常用 Markdown 编辑能力，不把页面变成完整 IDE。

轻量增强：最常用 5 个按钮

A. 保守增强

增加加粗、斜体、标题、列表、引用。改动小，风险低，但功能提升有限。

常用 Markdown 工具组

B. Typora 常用工具组

推荐：覆盖格式、标题、列表、任务、引用、代码、链接。功能明显增强，仍保持单行工具栏。

下拉分组 + 辅助面板

C. 分组下拉式

最接近成熟编辑器，可扩展表格、图片、目录、预览等，但首轮开发更重。

我建议先做 B

先把用户最常用的 Markdown 操作做好：选中文字后包裹语法，没有选区就插入模板；同时补快捷键 Ctrl+B / Ctrl+I / Ctrl+K。后续再加预览、表格和目录。
