# Verification Before Completion 中文说明

## 这个 skill 是什么

用于在声称任务完成、问题修复或测试通过前，强制先运行验证命令并读取结果。

## 什么时候使用

- 准备说“完成了”之前
- 准备说“测试通过”之前
- 准备提交、合并或创建 PR 前
- 修复 bug 后需要证明结果时

## 怎么使用

1. 明确什么命令能证明当前结论。
2. 运行完整验证命令。
3. 阅读输出和退出码。
4. 根据证据说明真实状态。

## 注意事项

- 不能用之前的运行结果代替当前验证。
- 不能只跑局部命令就声称整体通过。
- 失败时要如实说明失败内容。
