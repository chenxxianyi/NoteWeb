# Using Superpowers 中文说明

## 这个 skill 是什么

这是 Superpowers 技能集的入口说明，规定如何发现、选择和使用其他 skill，并强调在适用时必须调用相应 skill。

## 什么时候使用

- 每次会话开始时
- 不确定是否有 skill 适用时
- 需要理解 Superpowers 技能体系时
- 需要在不同 AI 平台之间映射工具名称时

## 怎么使用

1. 判断当前任务是否匹配某个 skill。
2. 如果匹配，先加载对应 skill。
3. 按 skill 指引执行任务。
4. 如果多个 skill 适用，选择最小必要集合。

## 注意事项

- 用户显式指令优先于 skill。
- skill 优先于默认工作习惯。
- 不同平台的工具名称可能不同，需要参考映射文档。
