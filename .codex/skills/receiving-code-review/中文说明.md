# Receiving Code Review 中文说明

## 这个 skill 是什么

用于处理代码评审反馈。它强调不要盲目接受，也不要情绪化回应，而是先理解、验证、评估，再逐项处理。

## 什么时候使用

- 收到代码评审意见
- 评审意见不够清楚
- 评审建议可能不适合当前代码库
- 需要逐项落实反馈

## 怎么使用

1. 完整阅读反馈。
2. 用自己的话理解每条意见。
3. 对照代码验证问题是否存在。
4. 判断建议是否技术上合理。
5. 一次处理一条，并运行验证。

## 注意事项

- 不要为了礼貌而假装反馈一定正确。
- 不清楚的反馈要先问清楚。
- 技术正确性优先于表面认同。
