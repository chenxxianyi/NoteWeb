# Finishing a Development Branch 中文说明

## 这个 skill 是什么

用于开发完成后的收尾阶段，帮助确认测试、检查环境，并决定合并、创建 PR 或清理分支。

## 什么时候使用

- 功能已经实现
- 测试已经通过或需要最终验证
- 准备合并分支
- 准备提交 PR
- 需要整理开发分支状态

## 怎么使用

1. 先运行项目测试。
2. 检查当前 Git 状态和分支情况。
3. 根据环境给出合并、PR、保留分支等选项。
4. 按用户选择执行后续操作。

## 注意事项

- 测试失败时不能进入合并或 PR。
- 不要自动替用户决定合并策略。
- 收尾前要确认没有遗漏的未提交改动。
